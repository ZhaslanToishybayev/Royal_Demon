# Gameplay Loop & Progression

## Core Loop
- Исследование процедурно генерируемых комнат, зачистка врагов и сбор ресурсов (золото, оружие, зелья).
- Улучшение арсенала через сундуки и дроп врагов; выбор оружия с уникальной механикой.
- Продвижение к босс-комнате, повышение сложности врагов и наград по мере удаления от стартовой точки.

## Прогрессия
- Сложность контролируется конфигом `config/difficulty.json`. Значения `enemyHealthMultiplier` и `enemyDamageMultiplier` масштабируют здоровье и урон врагов. Предусмотрены профили `easy`, `normal`, `hard`, можно расширять.
- Баланс оружия вынесен в `config/weapons.json`. Урон мечей читается при запуске и легко настраивается без пересборки.
