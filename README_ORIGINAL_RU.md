# Royal Demons
Royal Demons — это игра в жанре Dungeon Crawler, написанная на Java, которая выиграла конкурс Best Project Competition среди 114 команд на курсе Georgia Tech CS 2340 Objects and Design весной 2021 года.

## Предыстория:
*Королевство Короля когда-то жило в гармонии с различными типами существ в дикой природе. Стремясь к цели мирного сосуществования с дикой природой, Королевство Короля поставило задачу объединить людей и существ в одной среде. После векового существования королевства мира и процветания демонизированный человек вскоре осознал другой взгляд на этих существ — чтобы осуществлять свои жадные желания. Используя темную магию для одержимости этими существами и постепенного захвата королевства, злодей похищает существ одно за другим в свой подземелье и использует их против Королевства Короля. Прошло немного времени, и Демонический Человек приобрел свою новую армию когда-то счастливо живущих одержимых существ. После того, как похитили его лучшего друга, Динo Беби отправляется в путь мести, чтобы спасти свою девушку и восстановить равновесие в своем королевстве.*

## Особенности:
- Бесконечно играбельные карты
- Всего 9 различных видов оружия. Каждое имеет свою обычную и специальную атаку
- 20+ типов миньонов
- Финальный босс с 2 фазами
- Скрытая комната для вашего обнаружения
- Комната испытаний

## Управление игрой:
- WASD для перемещения героя
- Левый клик мыши для обычной атаки
- Пробел для мощной атаки
- E для подбора предметов
- F для использования предметов или взаимодействия с NPC
- I для показа инвентаря
- 1 для питья зелья здоровья
- 2 для питья зелья ярости
- M для показа карты игры
- P для телепорта в демонстрационную комнату

## Установка:
1. Клонируйте этот репозиторий
2. Откройте этот репозиторий в IntelliJ IDEA IDE
3. Должно появиться всплывающее окно с вопросом, хотите ли вы загрузить проект Gradle. Загрузите его
4. Установите SDK проекта на Amazon Corretto 11.0.20
5. Запустите приложение через Gradle! (Tasks -> application -> run)
